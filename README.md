# 🖼️ Website Screenshot Mockup Generator

A modern and simple tool built with HTML, CSS, and JavaScript to convert website screenshots into beautiful browser mockups.

---

## ✨ Features

- 📤 Upload website screenshot
- 🌐 Browser frame preview (Chrome, Firefox, Edge, Safari)
- 🎨 Realistic UI mockup design
- ⚡ Instant preview
- 💻 Fully responsive layout
- 🧠 Beginner friendly code

---

## 📂 Project Structure

```
website-screenshot-mockup/
│── index.html
│── style.css
│── app.js
│── README.md
```

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Upload your website screenshot
3. Select browser type
4. Click **Generate Mockup**
5. Preview your design instantly
